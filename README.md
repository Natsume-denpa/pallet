WhitePaletteは、白いPaletteに思い出の写真を載せていくことをコンセプトにしたツールです。<br>
1年分の思い出はPaletteでは耐えきれないほどの量になるため、リポジトリ名には「荷台」を意味する pallet を採用しています。

webでの使用:<br>
